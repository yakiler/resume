## 擅长方向
>性能优化：<br/>
>1.降低dom树节点层级嵌套<br/>
>2.优化js文件加载策略，首页面减轻体积，不必要的js文件采用懒加载。<br/>
>3.避免不正确的操作dom，需要加载的模块挂载入dom前完成相关属性的设置<br/>
>4.采用http2代替部分http<br/>
>5.尽量避免使用table布局<br/>
>6.多资源采用域名分片提升加载效率<br/>
>7.压缩/合并资源文件。小图标合并为大图减少http请求<br/>
>8.合理的使用缓存。减少网络加载和js计算<br/>
>9.合理的使用base64降低服务器压力<br/>
>10.合理的使用硬件加速<br/>
